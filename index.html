A propos

BASIRA est un jeu indépendant 2D de type RPG/Aventure développé avec le moteur Godot Engine.

  • BASIRA est mon premier jeu sur lequel je m'implique autant.
  • Je veux faire de ce jeu un jeu dans lequel le joueur est libre d'explorer le monde dans lequel il évolue.
  • Le joueur sera libre d'explorer le monde, mais aussi de participer à la construction de celui-ci ainsi que d'aider ses habitants.

Ce jeu est développé dans le cadre d'un projet personnel pendant ma formation professionnelle de développeur. Mais avant tout, c'est un projet qui me tient à coeur, car depuis mes premières années sur internet je rêve de pouvoir développer mon propre jeu vidéo. Aujourd'hui la chance se présente, et j'espère aller au bout de ce périple.
